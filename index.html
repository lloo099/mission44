Where reinforcement learning meets domestic AI silicon

This dashboard tracks three converging fronts: (1) the rapid evolution of RL post-training for reasoning LLMs (GRPO, RLVR and successors), (2) the maturing Huawei Ascend / NPU software–hardware stack, and (3) LLM modeling advances that matter on memory-constrained accelerators. The goal: help you find a concrete, fundable project at the intersection.

How to use this

  1. Skim each domain tab to map the landscape.
  2. Use tag filters + search to find adjacent work.
  3. Open Project Ideas for synthesized, feasibility-rated directions.
  4. Refresh Live Papers with the included script to stay current.

The core thesis

RL post-training is now the dominant lever for frontier capability, but almost all tooling assumes CUDA. Ascend NPUs offer abundant, increasingly available compute with a maturing stack (CANN, MindSpeed, vLLM-Ascend). The open opportunity is making efficient RL training actually work and scale on NPUs.
